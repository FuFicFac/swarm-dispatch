Premium models think.

Claude and Codex are best spent on architecture, decomposition, risk detection, inspection, and adjudication. That is where expensive reasoning changes the outcome.

Cursor builds.

Composer 2 or Cursor Agent CLI becomes the high-throughput implementation lane. It receives a bounded plan, edits assigned files, reports changed files, and hands work back for inspection.

The Pattern

PlanUse Claude/Codex to decide what should be built and where the risk is.
DispatchSend bounded implementation work to Cursor with a precise write set and stop condition.
InspectBring the result back to a stronger reviewer before accepting it.
RepeatKeep reassigning until the artifact is genuinely done.
