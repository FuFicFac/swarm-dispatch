# Codex Foreman Example

```text
Use $swarm-dispatch to ship a public documentation page for this repo.

Finish line:
The repo has a clean README, a GitHub Pages-ready site folder, and a privacy scan with no local paths or personal data.

Lanes:
1. Explorer: inspect current files and identify missing public docs.
2. Builder: create or update README and docs only.
3. Inspector: scan for private paths, names, credential material, and internal references.
4. Visual reviewer: open the Pages HTML locally and report layout issues.

Acceptance:
No private paths, no personal identifiers, no broken local links, and the site works from a static file server.
```
