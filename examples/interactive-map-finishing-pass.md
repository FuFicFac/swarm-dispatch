# Interactive Artifact Finishing Pass

```text
Use $swarm-dispatch for a finishing pass on an interactive HTML artifact.

Assign:
- Visual inspector: check responsive layout, contrast, text fit, and broken interactions.
- Content inspector: check claims, labels, and missing context.
- Privacy inspector: scan for local paths, personal names, internal project references, and credentials.
- Builder: apply only the accepted fixes after inspection.

Verify:
- Run a local static server.
- Open the page in a browser.
- Confirm no console errors.
- Re-run the privacy scan.
```
